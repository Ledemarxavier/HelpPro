# 📘 HelpPro - Sistema de Gestão para Profissionais Autônomos

HelpPro é uma aplicação web desenvolvida com **Spring Boot** e **MySQL**, com foco em facilitar a conexão entre clientes e profissionais autônomos. A plataforma permite que profissionais ofereçam seus serviços e que clientes agendem consultas ou atendimentos.
